# ForWork
## [CoderCustomSetup](CoderCustomSetup)
- Environment Automation: Quickly initialize a standardized personal workspace on Ubuntu-based servers.
- Comprehensive Neovim Setup: Automatically download and install the latest Neovim release, resolve all dependencies (gcc, ripgrep, nodejs, python, etc.), and clone the custom neovim-conf configuration.
- Enhanced Terminal Experience: Integrate Oh-My-Zsh with 3 productivity-boosting plugins (syntax-highlighting, autosuggestions, zsh-z) and apply the exclusive ngxxfus.zsh-theme.
- Safe Configuration Management: Automatically backup the existing .zshrc file, smartly migrate data to the new system, and integrate a familiar set of user aliases.
- Flexible Control: Designed with a modular approach, allowing you to easily enable or disable individual installation processes (Apt, Neovim, Oh-My-Zsh, Aliases) simply by toggling the 1 or 0 flags at the beginning of the script.
